# Task1
SQL query that identifies contiguous ranges of missing identifiers and returns these missing ID ranges as the result.

## Requirements
- [Python](https://www.python.org/downloads/) (version 3.11 or later)
- [Spark](https://spark.apache.org/downloads.html)/[PySpark](https://pypi.org/project/pyspark/) (version 3.5.4 or later)
- [Hadoop winutils](https://github.com/cdarlint/winutils) (for Windows)

## Usage
Run the `query.py` file as a regular Python script:
```
python query.py
```
